Structure definition (worldgen)
===============================

After you add your .nbt file to ../structures/, add a JSON here to register
the structure for world generation.

Example: mob_wrangler_house.json
  Defines the structure (size, biome placement, etc.) and points at your NBT.

See Minecraft wiki "Structure" and "Structure set" for JSON format, or use
datagen to generate the structure and structure_set JSONs.
